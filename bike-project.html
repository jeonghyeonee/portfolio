Seoul Public Bike Analyst

Business Analytics with ML | LightGBM + XGBoost

View Presentation

Summary

2023_BA_Project analyzes the Seoul Public Bike system to develop strategies that reduce financial deficits. The project predicts usage patterns using ensemble models to enhance system efficiency.

Development Period

  • Start: September 2023
  • End: December 2023

Platform & Tech Stack

  • OS: Windows, MacOS
  • Language: Python 3.11
  • IDE: Jupyter Notebook
  • Libraries: pandas, numpy, scikit-learn, LightGBM, XGBoost

Team

  • Lee Jeonghyeon: Data Preprocessing, ML Model, Ensemble
  • Chu Yulim: Financial Analysis, ML Model
  • Shin Dongwoo: Preprocessing, Optimization & Ensemble
  • Choi Junhyeok: Data Preprocessing, ML Model

Key Features

  • Usage prediction using LightGBM and XGBoost based on time and location
  • Identified key deficit factors through financial analysis
  • Achieved improved performance with ensemble modeling (RMSE 1.71, R² 0.647)

Troubleshooting

  • Issue: Handling missing values & encoding station IDs
  • Solution: Label encoding and data cleaning improved stability
  • Comparison: Ensemble model outperformed both LightGBM and XGBoost individually
