TrenD Catcher

Big Data × SNS × Recommendation

View Presentation

Summary

"Is it possible to find the backward linkage effect through Twitter tweets, the world's largest SNS?"

Trend Catcher analyzes industrial groups and products emerging due to popular dramas. By tracking Twitter trends, it recommends relevant products and predicts demand in connected industries.

Development Period

  • Start: February 2022
  • End: June 2022

Platform & Tech Stack

  • OS: Linux (AWS EC2)
  • Languages: Python, PHP, JavaScript
  • Frontend: HTML, JS, PHP
  • Backend: Hadoop, Hive, Sqoop, MySQL

Team

  • Lee Jeonghyeon: Data Processing and Integration
  • Choi Jina: Web UI & Development
  • Pi Yujin: Data Crawling & Preprocessing
  • Hwang Yujeong: Data Analysis & Visualization

Key Features

  • Twitter-based trend analysis for product and industry recommendation
  • Full-stack web development for visual insights
  • Big Data processing with Hadoop, Hive, and MySQL integration

Troubleshooting

  • Issue: Storage failures on local VMs using VMware
  • Solution: Migrated to AWS EC2 (5 nodes) for distributed services
  • Result: Increased scalability and reliability
