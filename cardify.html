Cardify

Secure Credit Card Validation with Homomorphic Encryption

View Presentation

Summary

Cardify is a Python web application that provides secure validation of credit card numbers and expiry dates using the PI-HEAAN library for homomorphic encryption. Users can input their credit card information through a user-friendly web interface, ensuring data privacy and security during the validation process.

Development Period

  • Start: February 2024
  • End: June 2024

Platform & Tech Stack

  • OS: Windows
  • Languages: Python, HTML, JavaScript
  • Framework: Flask + PI-HEAAN
  • Database: None (Real-time processing)

Team

Main Features

  • Credit card number validation using Luhn algorithm
  • Brand identification via homomorphic encryption
  • Expiry date validation using real-time comparison
