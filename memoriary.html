MEMORIARY

Assist Memory | Diary × AI × Health

View Presentation

Summary

Memoriary is an innovative app designed to support the elderly in preventing memory loss by combining diary, to-do list, and exercise features with cognitive quizzes. Through advanced technologies like OpenAI and Android sensors, it promotes mental and physical health in an aging society.

Development Period

  • Start: September 2023
  • End: December 2023

Platform & Tech Stack

  • OS: Android
  • Language: Kotlin
  • Database: RoomDB + Firebase
  • IDE: Android Studio
  • APIs: OpenAI GPT, DALL-E2, Android Sensors

Team

  • Lee Jeonghyeon: To-Do, Sign In, Nav Bar
  • Park Hyun Seo: Exercise, Quiz, My Page
  • Kim Siheon: Diary, Nav Bar

Main Features

  • Diary: Speech-to-Text input, DALL·E2 thumbnail generation
  • To-Do List: Calendar integration, task management
  • Exercise: “Shake It Up”, “Clapping” using sensors
  • Quiz: AI-generated memory quizzes based on diary
  • Membership: Sign in via password or fingerprint
  • My Page: Profile management and sign out

Contribution

  • Designed and implemented core app functions: diary, quizzes, and to-do list
  • Integrated APIs and ensured accessibility for elderly users
  • Led debugging, UI testing, and UX refinement

Troubleshooting

  • Issue: UI freezing due to database queries on main thread
  • Solution: Replaced LiveData with Observer pattern and suggested coroutines
  • Trade-off: Observer delayed sync but fixed blocking behavior
