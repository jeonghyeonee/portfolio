Memoriary

Memory-Support App for the Elderly

Summary

Memoriary is an innovative app designed to support the elderly in preventing memory loss by combining diary, to-do list, and exercise features with cognitive quizzes. Through advanced technologies like OpenAI and Android sensors, it promotes mental and physical health in an aging society.

Development Period

  • Start: September 2023
  • End: December 2023

Platform & Tech Stack

  • OS: Android
  • Language: Kotlin
  • Database: RoomDB + Firebase
  • IDE: Android Studio
  • APIs: OpenAI GPT, DALL·E2, Android Sensors

Team

  • Lee Jeonghyeon: To-Do, Sign In, Nav Bar
  • Park Hyun Seo: Exercise, Quiz, My Page
  • Kim Siheon: Diary, Nav Bar

Features

  • Diary: Voice-based input + DALL·E2 thumbnails
  • To-Do: Calendar + done list
  • Exercise: Shake/clap using Android Sensor API
  • Quiz: Daily GPT-based memory tests
  • Membership: Login via ID/PW or fingerprint
  • My Page: View/edit user profile

Troubleshooting & Solution

Faced issues with LiveData causing UI delays. Applied an Observer pattern and asynchronous operations using background threads to ensure a smooth user experience.
