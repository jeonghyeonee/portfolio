JiGu

Empowering Vegetarian Lifestyles

View JiGu Presentation

Summary

JiGu is a mobile application that helps users lead a vegetarian lifestyle with ease. It offers personalized recipes, nutritional information, restaurant searches, barcode food checks, chatbot assistance, and a community for vegetarians.

Features

  • Restaurant Finder: Locate vegetarian-friendly restaurants and explore detailed menus and ingredient info.
  • Personalized Recipes: Get ChatGPT-generated weekly meal plans tailored to your dietary needs and budget.
  • Nutritional Info: Check detailed nutritional values including vitamins, minerals, and protein content.
  • Barcode Scan: Use barcode scanning to check if a product fits your vegetarian type.
  • Community Board: Post and share recipes, experiences, and tips with fellow vegetarians.
  • User Profile: Kakao login integration and customizable characters for fun user interaction.

Repository Structure

  • drawable: Icons and image assets
  • layout: XML files defining each page layout
  • values: colors.xml and strings.xml
  • foodPage: Barcode-based food search
  • recipePage: Chatbot-based recipe suggestions
  • restSearch: Restaurant locator
  • MainActivity.kt: Core logic of the app

Branch Naming Rule

ID/Feature/IssueNum (e.g. jeonghyeonee/MakeMainPage/1)

How to Contribute

  1. Fork the repository
  2. Create a branch: git checkout -b ID/feature/issueNum
  3. Commit your changes
  4. Push to your fork
  5. Open a pull request
